# Task-Tracker-CLI-app

Task Tracker CLI is a Node.js-based command-line application that allows users to manage daily tasks efficiently. The application loads tasks from a JSON file, enables adding, removing, searching, and sorting tasks by priority or date, and calculates completion statistics using JavaScript functional methods such as map, filter, reduce, some, and every.

This project demonstrates strong understanding of JavaScript fundamentals, object-oriented programming concepts, built-in data structures, closures, file handling, and error management using try-catch.

## Features

- Load and save tasks using JSON file storage
- Add new tasks with priority and due date
- Remove tasks by ID
- Search tasks by title or status
- Sort tasks by priority or date
- Compute completion statistics using reduce()
- Check task status using some() and every()
- Private task counters implemented using closures
- CLI interaction via process.stdin
- Proper error handling using try-catch

## Tech Stack

- JavaScript
- Node.js
- File System Module (fs)
- JSON for data storage


## Learning Objectives

This project was built to strengthen understanding of:

- JavaScript data types and control flow
- Functional array methods (map, filter, reduce)
- Object manipulation and prototypal behavior
- Node.js runtime environment
- CLI input handling
- Error handling and validation
- Closure implementation for private state management
